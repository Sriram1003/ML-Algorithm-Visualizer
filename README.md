# 🧠 AlgoVista — Quantified Logic

> **Experience the raw math of artificial intelligence through interactive laboratory environments.**

AlgoVista is a high-fidelity machine learning visualization platform designed to de-mystify complex algorithms through real-time visual telemetry. Built with a "Neural Laboratory" aesthetic, it provides an immersive experience for students, researchers, and developers to explore the inner workings of ML models.

🚀 **[View Live Platform](https://sriram1003.github.io/ML-Algorithm-Visualizer/)**

![AlgoVista Banner](https://images.unsplash.com/photo-1677442136019-21780ecad995?auto=format&fit=crop&q=80&w=1200)

## 🧪 Interactive Laboratories

The platform is divided into specialized paradigms, each offering deep interactive simulations:

-   **Supervised Learning**: Neural architectures for pattern recognition including SVM, Regressions, and Decision Forest logic.
-   **Unsupervised Learning**: Clustering heuristics and dimensionality reduction kernels like K-Means, DBSCAN, t-SNE, and UMAP.
-   **Ensemble Methods**: Multi-layered model aggregation using Gradient Boosting (XGBoost) and Random Forest parallel processing.
-   **Reinforcement Learning**: Dynamic agent-based learning environments using Q-Learning and reward-based optimization.

## 🛠️ Technology Stack

AlgoVista is built using cutting-edge web technologies to ensure a high-performance, premium experience:

-   **Frontend**: [Next.js 13](https://nextjs.org/) (App Router)
-   **Language**: [TypeScript](https://www.typescriptlang.org/)
-   **Styling**: [Tailwind CSS](https://tailwindcss.com/)
-   **Animations**: [Framer Motion](https://www.framer.com/motion/)
-   **Visualizations**: [D3.js](https://d3js.org/), [Chart.js](https://www.chartjs.org/), [Recharts](https://recharts.org/)
-   **Mathematics**: [tsne-js](https://github.com/karpathy/tsnejs), [umap-js](https://github.com/paisley-software/umap-js)
-   **UI Components**: [Radix UI](https://www.radix-ui.com/)

## 🚀 Getting Started

To run the laboratory locally:

1. **Clone the repository**:
   ```bash
   git clone <your-repo-url>
   cd projectml/project
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Open the lab**:
   Visit [http://localhost:3000](http://localhost:3000) to start exploring.

## 📦 Deployment & Hosting

This project is optimized for static hosting. To create a production build:

```bash
npm run build
```

The static files will be generated in the `out/` directory, which can be hosted for free on:
-   **Vercel** (Official Recommendation)
-   **Netlify** (Pre-configured via `netlify.toml`)
-   **Cloudflare Pages**
-   **GitHub Pages**

## 🎨 Design Philosophy — "The Neural Laboratory"

AlgoVista uses a specialized design system called the **Neural Laboratory**:
-   **Palette**: Deep obsidian backgrounds (`#020202`) with vibrant purple and indigo accents.
-   **Glassmorphism**: Heavy use of backdrop-blur and semi-transparent borders for a high-tech "glass" feel.
-   **Typography**: Bold, black, italicized headers for a premium, aggressive scientific aesthetic.
-   **Motion**: Anticipated transitions and micro-interactions that make the data feel alive.
